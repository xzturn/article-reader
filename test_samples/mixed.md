# Python 入门指南

## 什么是 Python

Python 是一种广泛使用的**高级编程语言**。它以简洁、易读的语法著称，被称为"优雅"的编程语言。

Python 的创始人是 Guido van Rossum，他在 1989 年开始开发这门语言。更多信息可以参考[Python 官网](https://python.org)。

## 基础语法

### 变量和数据类型

Python 支持多种数据类型：

```python
name = "张三"
age = 25
height = 1.75
is_student = True
```

### 条件判断

使用 `if-else` 进行条件判断：

```python
if age >= 18:
    print("成年人")
else:
    print("未成年人")
```

## 数学运算

Python 可以进行各种数学运算。例如，圆的面积公式：

$$A = \pi r^2$$

以及一些常用的数学函数。

## 总结

Python 是一门非常值得学习的编程语言。无论是**数据分析**、*Web 开发*还是人工智能，Python 都有广泛的应用。
